# 🚀 Just Sudying

---

## 📖 Sobre o projeto

Este projeto nasceu com o objetivo de servir como um ambiente prático de aprendizado e revisão de conceitos modernos do ecossistema JavaScript/TypeScript.

Durante o desenvolvimento, estou explorando tópicos como:

* ⚡ Arquitetura Full Stack
* 🗄️ Integração com banco de dados
* 🎯 APIs REST
* 🧩 Componentização no React
* 🔄 Gerenciamento de estado
* 📦 Boas práticas de organização de código
* 🧪 Validações e tratamento de erros

Mais do que criar uma aplicação funcional, o foco principal é consolidar conhecimentos através da prática.

---

## 🛠️ Tecnologias utilizadas

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* NestJS
* Prisma ORM
* SQLite
